### Wiring RFID RC522 module
The following table shows the typical pin layout used:

| Signal    | MFRC522 | WeMos D1 mini |
| --------- | :-----: | :-----------: |
| RST/Reset |   RST   |      D3       |
| SPI SS    |   SDA   |      D8       |
| SPI MOSI  |  MOSI   |      D7       |
| SPI MISO  |  MISO   |      D6       |
| SPI SCK   |   SCK   |      D5       |
